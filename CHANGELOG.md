## [0.2.1]

### Added
- Writing editor with chapter/scene management
- TipTap-based rich text editor with Japanese typography (ruby, emphasis dots, auto-indent)
- Split view for editing two scenes side by side
- Version history with diff view and restore
- Knowledge base (character profiles, world-building, reference materials)
- Reference images for scenes
- Export to TXT, DOCX, PDF, and ePub
- Project-wide search and replace
- Focus mode and typewriter scroll
- Automatic and manual backup/restore
- Dark mode
- Multilingual support (Japanese and English)
