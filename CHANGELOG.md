## [1.0.0]

### Added
- Writing editor with chapter/scene management and drag & drop reordering
- Rich text editor with ruby (furigana), emphasis dots, and vertical-in-horizontal text support
- Chapter dashboard with corkboard view and manuscript editor
- Corkboard view for organizing scenes visually as cards
- Manuscript editor for editing multiple scenes at once within a chapter
- Split view for editing two scenes side by side
- Version history with diff view (side-by-side / unified) and restore
- Knowledge base for character profiles, world-building, locations, and reference materials
- Scene-to-knowledge linking with multi-select picker
- Annotations for adding text comments with inspector panel
- Scene synopsis editing
- Reference images for scenes with lightbox preview
- Export to TXT, DOCX, PDF, and ePub with advanced settings (page size, writing direction, ruby notation style, export mode)
- Export preview
- Project-wide search and replace
- In-editor search (Ctrl+F / Cmd+F)
- Focus mode
- Work statistics with real-time character count
- Editor footer with character count and selection tracking
- Font selection from system fonts
- Automatic and manual backup/restore
- Scene soft delete with recovery
- Dark mode
- Multilingual UI (Japanese / English)
- Help dialog with keyboard shortcuts and markup reference
- Persistent layout (panel sizes, inspector state, split view configuration)
